# Curatescape Classic
Responsive theme for Omeka sites using the Curatescape framework. Evolved from the first wave of Curatescape projects. Designed to accomodate a bold sitewide background image. 

## Companion Plugins
Use of [Curatescape Stories & Tours](https://github.com/CPHDH/Curatescape), [Curatescape SEO](https://github.com/CPHDH/CuratescapeSeo), and [Curatescape Galleries](https://github.com/CPHDH/CuratescapeGalleries) companion plugins is strongly recommended. 

## Additional Information
For a more modern theme, see: [Curatescape Echo](https://github.com/CPHDH/theme-curatescape-echo) theme.

For more information about Curatescape, see [curatescape.org](https://curatescape.org).
